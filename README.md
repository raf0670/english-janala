📝 English Janala (English Window)
English Janala is an interactive vocabulary-learning platform designed to help Bengali speakers master English words. It features categorized lessons, real-time search functionality, and integrated text-to-speech for pronunciation.

🚀 Live Demo
[English Janala](https://raf0670.github.io/english-janala/)

✨ Features
Categorized Learning: Words are organized into structured lessons (Levels) fetched dynamically from an external API.

Real-time Search: A powerful search bar that filters through the entire vocabulary database to find specific words instantly.

Text-to-Speech (TTS): Integrated Web Speech API to help users hear the correct pronunciation of English words.

Detailed Word View: Interactive modals providing in-depth information, including meanings, pronunciations, examples, and synonyms.

Responsive Design: Built with Tailwind CSS and DaisyUI to ensure a seamless experience across mobile, tablet, and desktop devices.

Dynamic UX: Includes a custom-built loading spinner and "No Data" states to improve user experience during API calls.

🛠️ Tech Stack
Frontend: HTML5, CSS3

Styling: Tailwind CSS, DaisyUI

Scripting: Modern JavaScript (ES6+)

API: Programming Hero Vocabulary API
